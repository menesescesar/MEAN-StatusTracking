## MEAN Production Line Status Tracking (Lite)

Simplistic status tracking project, using mean stack (angular 4).

### Installation

    npm install
    
Configure mongoDB connection in config-example.json and rename it to config.json.

### Usage

    npm start    
        
    cd angular-src
    ng build
    http://localhost:3000

### Tests

    npm test
